## 🏛️ Project Overview

**Aegis-Graph** is the industry's first **Sovereign Multi-Agent Framework** specifically engineered to safeguard academic integrity in the age of generative AI. Developed by the **Atlanta College of Liberal Arts and Sciences (ACLAS)**, it replaces manual visual inspection with a decentralized, graph-driven reasoning protocol.

By leveraging a massive indexed dataset of **102,482 global academic institutions (ROR nodes)**, Aegis-Graph provides real-time, high-fidelity verification of credentials, research outputs, and institutional affiliations.

---

## 🧠 System Architecture: The Agentic Swarm

Aegis-Graph operates via a triad of specialized AI agents that collaborate in a zero-trust environment:

### 1. 👁️ Vision Forensics Agent
Utilizes sub-pixel artifact analysis and CNN-driven texture verification to detect AI-generated or digitally manipulated credentials.
*   **Key Tech**: Discrete Wavelet Transform (DWT), Artifact Clustering.

### 2. 🗺️ Graph Navigator Agent
The core of our **GraphRAG** engine. It maps input data against the global institutional topology (ROR, OpenAlex, Crossref) to verify founding dates, temporal consistency, and spatial validity.
*   **Database**: 102,482 Registry Nodes.

### 3. ⚖️ Logic Auditor Agent
A Chain-of-Thought (CoT) reasoning engine that identifies logical paradoxes (e.g., a degree issued before a university's founding date) and scholarly footprint anomalies.
