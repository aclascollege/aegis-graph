## 🤖 Phase 3: The Agentic Swarm (Forensics, Navigator, Auditor)

Aegis-Graph orchestrates three specialized agents working in a recursive reasoning swarm:

1.  **Vision Forensics**: Analyzes visual entropy and pixel-level inconsistencies in credential artifacts.
2.  **Graph Navigator**: Real-time institutional mapping against ROR and OpenAlex knowledge graphs.
3.  **Logic Auditor**: Executes deterministic "Step-0" to "Step-N" reasoning to catch temporal and logical fraud.
