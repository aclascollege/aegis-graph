## 宖当 What is Aegis-Graph?

**Aegis-Graph** is the world's first open-source **Sovereign Academic Audit Protocol**. It uses **Agentic GraphRAG** and a federated swarm of specialized AI agents to perform deep logical verification of academic credentials 窶?far beyond what traditional OCR systems can achieve.

### Key Capabilities
- **Multi-Agent Forensics**: 3 specialized agents (Vision, Graph, Logic) collaborate in real-time.
- **Global Academic Graph**: Verification against 250M+ records via OpenAlex & ROR.
- **Zero-Knowledge Privacy**: PII never leaves the institutional edge.
- **85% Cost Reduction**: 3-tier compute cascade minimizes token expenditure.
