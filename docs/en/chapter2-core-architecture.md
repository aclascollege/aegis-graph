# Chapter 2: Core Architecture
> **"A Multi-Layered Defense Against Synthetic Academic Fraud."**

The Aegis-Graph architecture is built on the principle of **Defense-in-Depth**. Instead of a single monolithic model, it employs a **Distributed Intelligence Swarm** that operates across three distinct logic layers.

## 1. The Data Ingestion Layer
At the entry point, the system performs a **Multi-Spectral Normalization**. Whether the input is a high-resolution PDF or a scanned JPG, the system extracts two parallel streams:
*   **Visual Stream**: Passed to the *Vision Forensics Agent*.
*   **Semantic Stream**: Passed to the *Graph Navigator*.

## 2. The GraphRAG Reasoning Engine
Our proprietary GraphRAG (Graph Retrieval-Augmented Generation) engine is the core brain. It queries the **Sovereign Academic Graph (SAG)**, which contains:
*   **102,482 Institutional Nodes**.
*   **Historical Timeline Metrics** (Founding, Accreditation, Mergers).
*   **Geospatial Consistency Vectors**.

## 3. The Consensus Protocol
A final audit verdict is only reached when the agents achieve a **Consensus Quorum**. If the Vision agent flags a forgery but the Graph agent finds a valid institutional trail, the **Logic Auditor** performs a deep-dive reasoning chain to resolve the conflict.
