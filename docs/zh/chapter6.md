# 第6章：共识机制详解

Aegis-Graph 的最终审计结论是 **MARS 集群共识握手** 的结果。本章详细介绍智能体如何通过协议解决冲突并发布主权证明。

## ⚖️ 加权投票模型
每个智能体（视觉、图谱、逻辑）根据其证据强度贡献投票权重：
| 智能体 | 核心指标 | 基础权重 |
| :--- | :--- | :--- |
| **视觉 (VF)** | 凭证保真度 | 0.30 |
| **图谱 (GN)** | 机构地位 | 0.35 |
| **逻辑 (LA)** | 时间一致性 | 0.35 |

## 🤝 握手协议流程
1.  **并行审计**：智能体各自执行专项核验。
2.  **冲突检测**：如果 VF 标记高风险但 GN 发现高权威机构，将触发共识冲突。
3.  **思维链 (CoT) 仲裁**：逻辑审计员启动 CoT 推理，最终判定审计结论。
