# 第3章：智能体协作矩阵 (MARS)
**多智能体推理集群 (MARS)** 是 Aegis-Verify 协议的执行臂。本章详细介绍了每个代理的具体能力。

## 👁️ 代理 Alpha：视觉法证代理
负责物理层的核验。
*   **扩散检测**：识别 Stable Diffusion 或 Midjourney 等生成器产生的特有噪声模式。
*   **层级分析**：检测数字篡改，如印章或签名中的像素不一致。
*   **光学一致性**：核验字体粗细、字间距和墨迹渗透模拟。

## 🗺️ 代理 Beta：图谱导航代理
主权学术图谱 (SAG) 的“图书馆员”。
*   **ROR 集成**：直接查询全球研究机构注册表。
*   **OpenAlex 映射**：核验签发者的学术足迹。
*   **实体关联**：连接全球大学之间的母子公司/分支机构关系。

## ⚖️ 代理 Gamma：逻辑审计代理
审计过程中的“首席法官”。
*   **时间悖论核验**：确保学历日期不会早于学校创立日期。
*   **项目一致性**：核验该特定学位项目是否在该机构的历史认证范围内。
*   **思维链 (CoT)**：为最终审计裁决提供人类可读的合理解释。
