# 第2章：核心技术架构
> **"多层级防御，应对合成学术欺诈。"**

Aegis-Graph 的架构基于 **“深度防御”** 原则。它不依赖于单一的大模型，而是采用了一个跨三个逻辑层运行的 **“分布式智能体集群”**。

## 1. 数据摄取层
在入口处，系统执行 **“多光谱归一化”**。无论输入是高分辨率 PDF 还是扫描的 JPG，系统都会提取两个平行的流：
*   **视觉流**：传递给视觉法证代理。
*   **语义流**：传递给图谱导航代理。

## 2. GraphRAG 推理引擎
我们专有的 GraphRAG（图谱检索增强生成）引擎是核心大脑。它查询 **主权学术图谱 (SAG)**，其中包含：
*   **102,482 个机构节点**。
*   **历史时间线指标**（创立、认证、合并）。
*   **地理空间一致性向量**。

## 3. 共识协议
只有当智能体达成 **“共识法定人数”** 时，才会给出最终审计裁决。如果视觉代理标记了伪造，但图谱代理找到了有效的机构轨迹，**逻辑审计代理** 将执行深度推理链来解决冲突。
