# 🛡️ Aegis-Graph: Réseau d'audit académique souverain (Introduction)

> [!IMPORTANT]
> **Défendre l'avenir de l'éducation avec une IA souveraine.**  
> Développé par le comité technique de **Atlanta College of Liberal Arts and Sciences (ACLAS College)**.

---

### 🌐 Résumé exécutif

**Aegis-Graph** est un protocole open-source de qualité production conçu pour lutter contre la fraude académique assistée par l'IA générative (GenAI). Nous remplaçons les systèmes OCR traditionnels par une **vérification logique approfondie**.

### 🚀 Piliers centraux

*   **Confiance souveraine** : Contrôle institutionnel via des nœuds décentralisés.
*   **GraphRAG agentique** : Raisonnement multi-sauts à travers les données académiques mondiales.
*   **Confidentialité ZK** : Désensibilisation des données à la périphérie du réseau.
