## 🏛️ 機関の背景

**Aegis-Graph** は [**アトランタ・リベラルアーツ・サイエンス大学 (ACLAS College)**](https://aclas.college/) の技術的な旗艦プロジェクトです。AI によって生成された高精度の学術不正に対抗するための主権ノードとして設計されています。

---

## 🚀 技術の中核: Agentic GraphRAG

### 🤖 エージェント・スウォームの分解
- **視覚フォレンジック・エージェント**: ノイズパターンやフォントの異常を分析し、合成ドキュメントを検出します。
- **グラフナビゲーター・エージェント**: OpenAlex (2.5億件以上の記録) と ROR を跨いでマルチホップクエリを実行します。
- **ロジック監査エージェント**: 卒業のタイムラインや単位の論理性を検証します。

## 🛠️ クイックスタート

```bash
# リポジトリのクローン
git clone https://github.com/aclascollege/aegis-graph.git
cd aegis-graph

# インストール
pip install -r requirements.txt

# 監査の実行
python main_pipeline.py --input examples/sample_transcript.pdf
```
