## 🏛️ Contexte Institutionnel

**Aegis-Graph** est le fleuron technique du [**Atlanta College of Liberal Arts and Sciences (ACLAS College)**](https://aclas.college/). Il est conçu comme un nœud souverain pour lutter contre la fraude académique de haute fidélité générée par l'IA.

---

## 🚀 Noyau Technique : Agentic GraphRAG

Contrairement à la vérification OCR traditionnelle, Aegis-Graph vérifie la **Topologie Logique** via une cascade de calcul à 3 niveaux.

### 🤖 Analyse de l'Agent Swarm
- **Vision Forensics Agent** : Analyse les motifs de bruit et les anomalies de police pour détecter la génération synthétique.
- **Graph Navigator Agent** : Exécute des requêtes multi-sauts sur OpenAlex (250M+ enregistrements) et ROR.
- **Logic Auditor Agent** : Vérifie la chronologie des diplômes et la logique des crédits.

---

## 🔒 Sécurité et Confidentialité

Aegis-Graph implémente un modèle de sécurité **"Sovereign Edge"** :
*   **Nettoyage PII** : Les données personnelles sont supprimées au niveau du nœud institutionnel (NPU).
*   **Exécution RAM-Only** : Le traitement se fait en mémoire éphémère.

## 🛠️ Démarrage Rapide

```bash
# Cloner le nœud souverain
git clone https://github.com/aclascollege/aegis-graph.git
cd aegis-graph

# Installation
pip install -r requirements.txt

# Lancer l'Audit
python main_pipeline.py --input examples/sample_transcript.pdf
```
