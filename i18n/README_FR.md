# 🛡️ Aegis-Graph : Protocole d'audit académique souverain

> **"Défendre l'avenir de l'éducation avec une IA souveraine."**  
> Un protocole open-source de qualité production pour la vérification logique approfondie des titres académiques, conçu par le comité technique de **Atlanta College of Liberal Arts and Sciences (ACLAS College)**.

---

## 🏛️ Fondement Institutionnel

**Aegis-Graph** est conçu par l'équipe d'ingénierie d'ACLAS College, fonctionnant comme un nœud souverain dans le réseau mondial de confiance académique.

- **Site officiel** : [aclas.college](https://aclas.college/)
- **Contact technique** : [info@aclas.college](mailto:info@aclas.college)
- **Dépôt GitHub** : [github.com/aclascollege/aegis-graph](https://github.com/aclascollege/aegis-graph)

---

## 🚀 Architecture de pointe
Utilisant **Agentic GraphRAG**, Aegis-Graph combat la fraude académique assistée par l'IA générative.
